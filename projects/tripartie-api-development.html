API Development for Tripartie Fintech Application

Started February 2018
Duration: 1 month
Completed

As a freelance developer, I collaborated with Tripartie, a fintech startup focused on secure transactions and dispute resolution. Working closely with the CTO and a small team, I developed the Symfony-based API for their payment escrow system, created controllers to manage accounts and transactions, and implemented Doctrine repositories for data retrieval. Additionally, I was responsible for bug fixes and writing specifications.
