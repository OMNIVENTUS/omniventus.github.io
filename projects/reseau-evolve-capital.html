Challenges & Solutions

Challenges

  • 1Building and deploying an SEO-optimized static website on a limited budget.
  • 2Ensuring the site remains lightweight and fast while integrating dynamic elements like animations.
  • 3Using AI-assisted tools efficiently to speed up development without sacrificing quality.

Solutions

  • 1Leveraged Next.js Static Site Generation (SSG) to generate a fast and cost-effective website.
  • 2Hosted the site on GitHub Pages to minimize hosting costs.
  • 3Used Framer Motion for lightweight yet engaging animations.
  • 4Utilized Cursor AI editor for efficient development, reducing build time to 15 hours.
