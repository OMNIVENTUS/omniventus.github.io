Réseau Evolve Capital Website

Started January 2025
Duration: 3 days
Completed

This website was built to enhance the online presence of Réseau Evolve Capital, a business and investment networking association. It aims to attract new members, increase traffic, and facilitate sponsorship acquisition. The site is fully static, leveraging Next.js with Static Site Generation (SSG) for optimal performance and cost efficiency by hosting on GitHub Pages. It includes privacy-first analytics using Cloudflare Analytics, dynamic animations with Framer Motion, and AI-assisted content generation for a streamlined workflow.
